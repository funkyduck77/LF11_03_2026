Für manuelles Einbinden von JUnit 6 brauchst du **diese 4 Dateien**:
<https://mvnrepository.com/artifact/org.junit.platform/junit-platform-engine/6.0.0>
<https://mvnrepository.com/artifact/org.junit.jupiter/junit-jupiter-api/6.0.0>
<https://mvnrepository.com/artifact/org.junit.jupiter/junit-jupiter-engine/6.0.2>
<https://mvnrepository.com/artifact/org.junit.platform/junit-platform-commons/6.0.2>

| Datei                              | Download |
|------------------------------------|----------|
| `junit-jupiter-api-x.x.x.jar`      |          |
| `junit-platform-engine-x.x.x.jar`  |          |
| `junit-jupiter-engine-x.x.x.jar`   |          |
| `junit-platform-commons-x.x.x.jar` |          |

Lade die neuesten Versionen herunter (.jar) (z. B. 2.17.x) und füge sie deinem Projekt hinzu:

**In IntelliJ / Eclipse:**

- Rechtsklick auf dein Projekt
- „Open Module Settings"/Project Structure (IntelliJ)
- Libraries
- add Java
- Die vier Dateien auswählen
